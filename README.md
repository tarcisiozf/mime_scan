# Mime-Type Scanner
Scan files and remove not allowed mime types

# How to use
Run on terminal:
```
./mime.py /path/to/folder /path/to/allowed_list
```

# Allowed list format
Must be separated by line break, example:
```
image/png
video/mp4
```
Mime-Type list:
http://hul.harvard.edu/ois/systems/wax/wax-public-help/mimetypes.htm
